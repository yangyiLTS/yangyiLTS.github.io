给树莓派上的OSMC安装简单桌面和Chromium

发表于 2018-01-23 | 分类于 笔记 |
字数统计: 209

OSMC是一个可以用于树莓派的多媒体应用系统，其中集成的kodi可以很好的作为一个家庭多媒体中心，但是OSMC默认并没有集成浏览器功能，我在OSMC论坛上找到一个为OSMC安装Chromium的脚本。

来源：Joakim_Sandstrom — [Testing] X11(simple desktop) + Chromium(web browser) in a simple installer for Stretch

X11是一个用于OSMC系统的简单桌面，包含了Chromium和文件管理功能。

安装：

wget https://raw.githubusercontent.com/zjoasan/x11-osmc/master/install_x11.sh
sh ./install_x11.sh

大约需要10分钟，最终在OSMC内添加一个启动插件，依次进入Settings->Add-on Browser->My add-ons->Program add-ons->X11 Launcher->Enable，启动X11桌面。

至于在线视频，原作者说

Chromium is not accelerated, so streaming isn’t the goal of this installation.

实测B站h5播放器可以运行，10秒后卡死。
